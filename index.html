🤖 Jira Auto-Update Hook

This project demonstrates automatic Jira ticket updates triggered by Git commits and pushes.

How It Works

1
Write a commit with a Jira ticket ID:
2
Commit is validated — rejected if no ticket ID present
3
An LLM-generated summary is posted as a comment on the Jira ticket
4
On git push, the ticket transitions to Review

Benefits

  • No more manuallysss updating Jira tickets
  • Sprint boards stay accurate automatically
  • Reviewers get context without reading the full diff
  • Every commit is traceable to a ticket
